# Лабиринты

## Описание

Консольная программа для генерации лабиринтов и поиска пути в них. Поддерживает несколько алгоритмов генерации и решения, вывод в консоль или файл, ASCII и Unicode-рендеринг.

## Функциональные требования

- Генерация лабиринтов алгоритмами DFS, Prim, Random Walk.
- Поиск пути алгоритмами A*, Dijkstra, SPFA.
- Ввод параметров через CLI (ширина, высота, алгоритм, начальная и конечная точки).
- Отображение лабиринта и найденного пути в консоли.
- Сохранение результата в файл.

## Формат файла с лабиринтом

| Символ | Значение |
|---|---|
| `#` | стена |
| ` ` | проход |
| `.` | точка маршрута |
| `O` | начальная точка |
| `X` | конечная точка |

Пример:

```
#####################
#O#   #  ...........#
#.### # #.#########.#
#...#   #.#       #.#
###.#####.# ### # #.#
#...#...#.#   # # #.#
#.###.#.#.### # # #.#
#.#  .#.#...# # # #.#
#.###.#.###.# # ###.#
#.....#.....# #  X..#
#####################
```

## Black-box тесты

Тест-кейсы находятся в `test/cases/`. Инструкция по запуску: [test/README.md](test/README.md).

| Сценарий | Описание |
|---|---|
| `1_check_help` | Проверка вывода `--help` |
| `2_check_point_validation` | Валидация координат точки `--start` |
| `3_check_maze_generation` | Генерация и сохранение лабиринта 1×1 |
| `4_check_maze_solving` | Решение заданного лабиринта алгоритмом A* |
| `5_check_optimal_path_choice` | Проверка выбора оптимального маршрута |
